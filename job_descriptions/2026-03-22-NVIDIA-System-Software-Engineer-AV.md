**System Software Engineer, Calibration – Autonomous Vehicles**
Company: CH01 NVIDIA Switzerland AG
Location: Zurich, Switzerland
Job ID: JR2014865
Posted: March 18, 2026

Responsibilities include developing and validating calibration algorithms, maintaining existing approaches, supporting infrastructure monitoring, and collaborating with OEM partners on prototype and production vehicle features.

Specific tasks:
- Creating sensor calibration approaches using classical and deep learning methods
- Optimizing software frameworks for customer requirements
- Working with diverse sensors (camera, LIDAR, radar, IMU, GPS)
- Writing production C++ and CUDA code meeting automotive safety standards
- Developing fleet monitoring tools and conducting vehicle testing
- Adapting calibration algorithms for new vehicle variants

**Required Qualifications:**
- PhD with 1+ years relevant experience, OR Master's with 3+ years, OR Bachelor's with 5+ years
- Excellent C++ and Python programming skills
- Knowledge of robotics, controls, estimation, and 3D computer vision
- Strong knowledge of programming and debugging for automotive/robotics systems
- Linux, Git, and build system proficiency

**Preferred Qualifications:**
- CUDA and data-parallel programming experience
- Performance analysis and optimization expertise
- ADAS and automotive systems background
- Bazel, Docker, and code analysis tool experience
- 4+ years calibration project experience
