## Full Job Description Text

**Lead Data Engineer (Databricks, PySpark)**
Company: EPAM Systems
Location: London, United Kingdom (Hybrid)
Department: Data Software Engineering

The role focuses on designing and delivering next-generation data platforms. The candidate will develop scalable ETL workflows and data pipelines using Databricks and cloud technologies, driving migration toward a Lakehouse architecture while collaborating with cross-functional teams in an agile setting.

**Key Responsibilities:**
- Architect and maintain production-grade data applications using Databricks, PySpark, Python, and Scala
- Lead modernization efforts toward Lakehouse architecture with Delta Lake and Unity Catalog
- Optimize Spark performance, manage data skew, and enhance query execution efficiency
- Champion software engineering standards including CI/CD pipelines and Infrastructure as Code
- Build secure, scalable solutions integrating Databricks with AWS, Azure, or GCP services
- Design AI-driven solutions incorporating LLMs and machine learning model operationalization
- Mentor engineering teams and translate complex technical concepts to stakeholders

**Core Requirements:**
- Bachelor's or Master's degree in Computer Science or related field
- Deep proficiency in PySpark with advanced performance tuning expertise
- Extensive production experience with Databricks Workflows, Delta Lake, and Unity Catalog
- Demonstrated Lakehouse architecture design and migration experience
- Strong Python skills (Scala beneficial) with OOP and functional programming knowledge
- Proven CI/CD, Infrastructure as Code, and Git version control implementation
- Senior/Lead-level experience driving technical strategy and architectural decisions
- Experience with automated testing frameworks and data quality validation

**Preferred Qualifications:**
- Official Databricks certifications (Data Engineer Professional, Spark Developer)
- Hands-on experience with AI/Agentic workflows, LangChain, or LlamaIndex frameworks
